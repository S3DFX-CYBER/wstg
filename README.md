# OWASP Web Security Testing Guide (WSTG) – हिंदी संस्करण

OWASP® Web Security Testing Guide (WSTG) के आधिकारिक रिपॉजिटरी में आपका स्वागत है।  
WSTG एक पूरी गाइड है जो वेब एप्लिकेशन और वेब सेवाओं की सुरक्षा जांच (Security Testing) के लिए उपयोग की जाती है।

यह गाइड दुनिया भर के सुरक्षा विशेषज्ञों और स्वयंसेवकों के सहयोग से बनाई गई है।  
पेनिट्रेशन टेस्टर्स, डेवलपर्स और संस्थाएँ इसे वेब सुरक्षा की best practices के लिए उपयोग करती हैं।

## WSTG टेस्ट को कैसे Refer करें

हर टेस्ट का एक यूनिक आईडी होता है:

`WSTG-Category-Number`

उदाहरण:  
`WSTG-INFO-02` → Information Gathering का दूसरा टेस्ट

बेहतर तरीका है Version के साथ लिखना:

`WSTG-v42-INFO-02`

इससे पता चलता है कि यह Version 4.2 का टेस्ट है।

> नोट: अगर Version नहीं लिखा है, तो उसे नवीनतम (latest) माना जाएगा।
